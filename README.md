Prerequisites

Node.js >= 16.x
npm atau pnpm

Installation
# Clone repository
git clone https://github.com/meilaa221/ndi-dashboard-test

# Masuk ke folder project
cd ndi-dashboard-test

# Install dependencies
npm install

# Jalankan development server
npm run dev

# Buka browser
http://localhost:3000


Build Production

npm run build
npm start
